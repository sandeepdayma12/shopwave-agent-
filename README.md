ShopWave: Autonomous Support Resolution Agent
Submission for the Agentic AI Hackathon 2026
ShopWave is an enterprise-grade agentic pipeline designed to automate high-volume customer support workflows. Unlike traditional chatbots, ShopWave utilizes a ReAct (Reasoning + Acting) framework powered by LangGraph to autonomously classify, investigate, and resolve customer tickets with a human-like audit trail.
🚀 Key Innovation Features
Stateful Orchestration: Built on LangGraph, the agent maintains a complex state across multiple reasoning loops, ensuring it doesn't get stuck in repetitive cycles.
Concurrent Ticket Processing: A custom asynchronous worker pool processes multiple tickets in parallel, significantly reducing latency for batch operations.
Intelligent Tool Calling: Features a sophisticated "Nudge" mechanism. If the agent fails to find a solution, the system identifies missing data and prompts the LLM to utilize specific tools (e.g., Knowledge Base vs. Order Database).
Resiliency & Logging: Includes a Dead-Letter Queue (DLQ) for tickets requiring human intervention and a comprehensive Audit Trail for every internal thought and action.
🛠 Technical Architecture
Core Logic: Python 3.10+
Orchestration: LangGraph & LangChain (ReAct Loop)
Intelligence: Llama-3.1-70b (via OpenRouter)
API Layer: FastAPI (High-performance asynchronous endpoints)
Interface: Streamlit (Real-time monitoring and trace inspection)
Environment: Dockerized for seamless deployment
⚙️ Environment Configuration
To run the agent, you will need an API key from OpenRouter.
Clone the Repository
Set Up Environment Variables:
Create a .env file in the root directory and add your key:
code
Bash
OPENROUTER_API_KEY=your_key_here
OPENROUTER_MODEL=meta-llama/llama-3.1-70b-instruct
🏃 Deployment Guide
Option A: One-Command Setup (Recommended)
The fastest way to evaluate the project is via Docker Compose, which spins up the API and the UI simultaneously.
code
Bash
docker compose up --build
Dashboard (Streamlit): http://localhost:8502
Interactive API Docs: http://localhost:8007/docs
Option B: Local Development
If you prefer running the processes manually:
1. Start the FastAPI Server:
code
Bash
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --host 0.0.0.0 --port 8007
2. Launch the Streamlit Dashboard:
code
Bash
streamlit run app.py --server.port 8502

scripts/: Utility scripts for visualizing the agent's architecture.
Security Note
This project is built with security-first principles. API keys are managed via environment variables and are excluded from version control via .gitignore. Please ensure any rotated keys are updated in your local .env before testing.
