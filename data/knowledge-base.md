# ShopWave Support Knowledge Base

## Return Policy
- Most products have a **30-day return window**.
- **Electronics accessories**: 60-day return window.
- **High-value electronics**: 15-day return window only.

## Refund Policy
- Refunds are only issued after eligibility is confirmed.
- VIP customers have extended leniency for returns.

## Warranty Policy
- Warranty claims go to the warranty team. Agents should escalate these.

## Wrong or Damaged Items
- Damaged on arrival: Full refund without requiring a return.
- Wrong item: Free return or exchange.
